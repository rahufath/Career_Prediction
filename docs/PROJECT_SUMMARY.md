# 📊 Project Summary

## Executive Overview

The AI Career Guidance System is a complete machine learning solution that predicts optimal IT careers based on emotional patterns and personality traits. The system achieves **~75% accuracy** on career prediction and provides natural language explanations for its recommendations.

### Feature Engineering

| Feature | Formula |
|---------|---------|
| Dominant Emotion | argmax(emotions) |
| Emotion Entropy | -Σ(p × log(p)) |
| Emotion Variance | var(emotions) |
| Neutral Ratio | neutral value |
| Positive Ratio | happy + surprise |
| Negative Ratio | angry + disgust + fear + sad |
